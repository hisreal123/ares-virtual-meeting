# Ares Virtual Meeting

Microsoft Teams–style virtual meeting UI (join page + in-call meeting page).

**Stack:** React, Vite, Tailwind CSS, shadcn/ui

See [REQUIREMENTS.md](./REQUIREMENTS.md) for scope, checklist, and progress.

## Development

```bash
bun install
bun run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Environment variables

```env
# Meeting page — number of participant tiles (coming soon)
VITE_PARTICIPANT_COUNT=6
```
